# Our Story - A Romantic Interactive Experience

A modern, cinematic storytelling website built with React, Vite, TailwindCSS, and Framer Motion.

## Features

- **Immersive Animations**: Smooth transitions and scroll-triggered reveals using Framer Motion.
- **Responsive Design**: Mobile-first layout for sharing on any device.
- **Interactive Elements**: Typing effects, confetti celebrations, and floating music player.
- **Themed Styling**: Dark romantic aesthetic with deep purples and soft pinks.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Customization

- **Story Content**: Edit `src/data.js` to add your own timeline events and letter content.
- **Images**: Replace placeholder image URLs in `src/data.js` with your own photos.
- **Music**: Update the audio source in `src/components/MusicPlayer.jsx`.

## Tech Stack

- **React 19**
- **Vite**
- **TailwindCSS v4**
- **Framer Motion**
- **Lucide React** (Icons)
- **Canvas Confetti**
